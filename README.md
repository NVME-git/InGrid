# InGrid

A cross-platform Sudoku app built with Flutter. Features Ingrid — an offline coaching engine using human-style techniques — plus real-time multiplayer with guess detection, and full solve replay with key move highlighting.

---

## Getting started

### Prerequisites

| Tool        | Minimum version      |
| ----------- | -------------------- |
| Flutter SDK | 3.11.3               |
| Dart SDK    | bundled with Flutter |

### Run

```bash
# Get dependencies
flutter pub get

# Run on connected device / browser
flutter run                  # picks a connected device
flutter run -d chrome        # web
flutter run -d macos         # macOS desktop
```

### Build

```bash
flutter build web            # production web bundle → build/web/
flutter build apk            # Android APK  → build/app/outputs/apk/
flutter build ios            # iOS (requires macOS + Xcode)
flutter build macos          # macOS app bundle
```

## Architecture overview

```
lib/
├── app.dart                  # MaterialApp.router, GoRouter config, theme
├── main.dart                 # Entry point, ProviderScope
├── core/
│   ├── engine/               # Pure Dart Sudoku engine (generator, validator, board)
│   └── replay/               # Move replay recorder
├── features/
│   ├── game/                 # Game screen, home screen, state (Riverpod), widgets
│   │   └── widgets/          # SudokuGrid, DigitToolbar
│   ├── history/              # Completed-game history screen
│   ├── stats/                # Per-difficulty statistics screen
│   ├── help/                 # How-to-play screen
│   ├── import/               # Manual puzzle import
│   ├── multiplayer/          # Real-time multiplayer (stub/Supabase)
│   └── profile/              # Player profile (stub)
└── services/
    ├── persistence_service.dart   # SharedPreferences-backed history & stats
    ├── session_service.dart       # In-memory game session guard
    └── theme_notifier.dart        # Light / dark mode Riverpod provider
```

State management uses **Riverpod** (`flutter_riverpod`). Navigation uses **GoRouter**.

## Features

- **Four difficulties** — Easy, Medium, Hard, Extreme (generated by the built-in engine)
- **Entry modes** — Number, Corner pencil-mark, Centre pencil-mark, Cell colour highlight
- **Multi-select** — Drag or long-press to select multiple cells and fill them at once
- **Undo / Redo** — Full move history
- **Auto-candidates** — Computed pencil-marks for every empty cell (toggle)
- **Conflict highlight** — Flags cells that violate Sudoku rules
- **Pause / resume** — Hides the grid and pauses the timer
- **Landscape layout** — Side toolbar with a toggle to move it left or right
- **Solve history** — Browse and export past games
- **Statistics** — Best time, average, last-7-day trend chart per difficulty
- **Multiplayer** — Real-time competitive play via Supabase (in progress)
- **Dark / light theme** — Persisted across sessions

## Keyboard shortcuts

| Key                    | Action                      |
| ---------------------- | --------------------------- |
| `1` – `9`              | Enter digit in current mode |
| `N`                    | Switch to Number mode       |
| `V`                    | Switch to Corner note mode  |
| `C`                    | Switch to Centre note mode  |
| `M`                    | Toggle multi-select         |
| `Z`                    | Undo                        |
| `Y`                    | Redo                        |
| `P`                    | Pause / resume              |
| `Delete` / `Backspace` | Erase selected cell(s)      |
| `Escape`               | Deselect all cells          |
